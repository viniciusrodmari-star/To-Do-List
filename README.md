# 📌 Minha To-Do List

Um sistema de gerenciamento de tarefas desenvolvido com **HTML, CSS e JavaScript**, com foco em organização pessoal e produtividade.
O projeto foi construído com base em conceitos de **Engenharia de Prompt aplicada ao desenvolvimento de software**, evoluindo de um To-Do simples para uma solução mais completa com agenda integrada.

---

## 🚀 Funcionalidades

### ✅ Gerenciamento de Tarefas

* Adicionar tarefas com:

  * Título
  * Descrição
  * Data e horário
  * Prioridade (Alta, Média, Baixa)
* Marcar tarefas como concluídas
* Remover tarefas
* Persistência de dados com **LocalStorage**

---

### 🗓️ Agenda (Calendário)

* Visualização mensal
* Navegação entre meses (anterior / próximo)
* Destaque do dia atual
* Exibição de tarefas diretamente no calendário com:

  * Horário
  * Nome da tarefa
* Clique no dia → abre detalhes com:

  * Título
  * Descrição
  * Prioridade
  * Horário
* Ordenação automática por horário

---

### ⏰ Controle de Tempo Inteligente

* Exibição dinâmica:

  * “Faltam X minutos”
  * “Faltam X horas”
  * “Faltam X dias”
  * “Já passou”
* Identificação automática:

  * 🔴 Tarefas vencidas
  * 🟡 Tarefas próximas do prazo (até 1h)

---

### 🎨 Estilização Condicional

* Cores diferentes para prioridade:

  * 🔴 Alta
  * 🟠 Média
  * 🔵 Baixa
* Destaques visuais:

  * Tarefa concluída (verde e riscada)
  * Tarefa vencida (vermelho)
  * Tarefa próxima (amarelo)

---

### 💬 Sistema de Feedback (UX)

* Mensagens dinâmicas (Toast):

  * 🟢 Sucesso
  * 🔴 Erro
  * 🟡 Aviso
* Feedback ao:

  * Criar tarefa
  * Atualizar tarefa
  * Remover tarefa
* Validação de campos obrigatórios
